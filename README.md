# Dog Album React

## Description
The **Dog Album React App** is a simple beginner-friendly React project that displays a dog image along with a heading. The app uses a functional component with hardcoded data, making it easy to understand **basic React component structure, JSX, and importing images**.  

This project is perfect for beginners who are learning how to create and render React components.

---

## Features
- Displays a static heading: “Dog Album”
- Shows a dog image imported from local assets
- Simple functional React component
- Easy to understand and modify
